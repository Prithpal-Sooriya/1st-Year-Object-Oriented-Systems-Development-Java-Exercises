The MyInteger class

create a class called MyInteger
class contains:

-->int value, stores a value given
-->constructor that creates MyIntegerObject for a specified int
-->get method for value
-->methods isEven(), isOdd(), isPrime() return true/false
-->static methods isEven(int), isOdd(int), and isPrime(int) return true/false
-->methods equals(int) and equals(MyInteger, true or false
-->static method parseInt(char[]) converts char array to number
--> static method parseInt(String) converts String to number
