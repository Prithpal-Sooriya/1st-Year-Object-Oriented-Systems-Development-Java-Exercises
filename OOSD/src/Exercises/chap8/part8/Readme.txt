Create UML and Java class of Fan

-int constants SLOW MEDIUM FAST (1, 2, 3) fan speed
-int speed (default SLOW)
-boolean on whether the fan is on (default false)
-double radius, radium of the fan (default 5)
-String colour, colour of fan (default blue)

-accessor and mutator methods for all of the data fields

-no arg constructor that creates a default fan
-method toString() that returns the description of fan
    -if on, then displays the fan speed, colour, radius
    -if off, then displays the colour, radius, "fan is off"
